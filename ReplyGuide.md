# ins-auto-reply-script Documentation

## Introduction to Auto-Reply Feature

MadHub's powerful auto-reply capabilities streamline the process of managing social media interactions by automatically detecting and responding to incoming messages. This feature is particularly valuable in maintaining consistent communication with users on platforms like Instagram, ensuring that no message goes unnoticed. With its robust automation tools, MadHub enhances user engagement and operational efficiency, making it an indispensable asset for any social media management strategy.

## Feature Capabilities

The **Instagram Auto-Reply** feature allows for:
- **24/7 Automated Responses**: Ensures timely replies to messages, enhancing user satisfaction and engagement.
- **Multiple Response Modes**: Offers both input mode (predefined responses) and file mode (loading replies from a file), providing flexibility in managing replies.
- **Configurable Parameters**: Users can set various operational parameters, including response intervals and the duration of running sessions, enabling tailored responses based on user engagement patterns.

These capabilities empower users to manage their accounts more effectively, reduce manual workload, and ensure consistent communication with followers.

## Usage Instructions

To implement the Instagram Auto-Reply feature within MadHub, follow these steps:

1. **Set Up MadHub**:
   - Install MadHub on your preferred platform (real Android device, cloud phone, or emulator).
   - Ensure you have logged into your Instagram account through the MadHub interface.

2. **Access Auto-Reply Settings**:
   - Navigate to the "Instagram" section within the MadHub dashboard.
   - Select the **Instagram Auto-Reply** feature from the available tools.

3. **Configure Response Settings**:
   - Choose your **Response Mode**:
     - **Input Mode**: Directly enter predefined messages.
     - **File Mode**: Upload a text file containing your responses.
   - **Set Operation Intervals**: Specify how often the tool should check for unread messages. For instance, you can set an interval of every 5 minutes.
   - **Define Single Run Duration**: Decide how long the auto-reply feature should operate continuously, e.g., for 3 hours before stopping.

4. **Activate the Feature**:
   - Once all configurations are complete, activate the auto-reply by clicking the "Start" button. This will initiate the monitoring and replying process.

5. **Monitor Responses**:
   - Regularly check the dashboard to ensure that the auto-reply is functioning as expected and adjust settings as necessary based on engagement feedback.

## Configuration Example

To effectively utilize the Instagram Auto-Reply feature, here is an example of how to set it up:

```plaintext
1. Response Mode: Input Mode
   - Response Messages: 
     - "Thank you for reaching out! We'll get back to you shortly."
     - "Hi there! Your message is important to us."

2. Operation Interval: 5 minutes
3. Single Run Duration: 3 hours
```

By configuring the auto-reply with these parameters, you can ensure that your followers receive prompt responses, enhancing their overall experience while optimizing your engagement efforts.
